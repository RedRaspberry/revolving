# List of Disabled Features

This list may not be comprehensive, if there are any omissions please let me know on discord. Please at least skim this list before playing.
# Vanilla Mechanics
The following features have been disabled or adjusted for balance:
- Torches now burn out.
- Certain other light source blocks emit a weaker light level than they originally did.
- Animals will drop more meat if kept in good conditions.

# Immersive Engineering Multiblocks
| Multiblock  | Suggested Alternative |
| ------------- | ------------- |
| Mixer  | Depends on recipe- moved to TFMG Chemical Vat or Create Mixer  |
| Metal Press  | Use Create's Press instead  |
| Automated Engineer's Workbench  | Moved to Sequenced Assembly or Mechanical Crafting |
| Excavator  | Use the **Create: Rock And Stone** drill bearing instead |
| Blast Furnace, Adv Blast Furnace  | Steel is produced another way, check EMI  |
| Refinery  | Recipes moved to Chemical Vat  |
| Sawmill  | Use Create's Sawmill instead  |
| Alloy Smelter  | Removed for balancing, check EMI for alternatives for your specific alloy |
| Assembler | Use Create's Mechanical Crafting instead |

# Electrical Consumers and Producers
The following features have been disabled because they circumvent the electrical systems present in the **Create: Power Grid** and **Create: Electro Energistics** mods.
- All IE Wire Connectors, Wires
- IE Electric Lamps
- All TFMG Wire Connectors, Wires, Motors, Cables, Cable Hubs, Etc

# Electrical/Rotary Power Generation
The following features have been disabled because of content overlap with **Create: Diesel Generators**,  **Create: Power Grid** and **Create: Electro Energistics**:
- **Create: The Factory Must Grow** Rotor, Stator, Generator
- All engines and turbines from **Create: The Factory Must Grow**

# Oil Processing
The following features have been disabled because of content overlay with **Immersive Petroleum**: 
-  **Create: The Factory Must Grow** Pumpjack Base and all related components, Oil Deposits, and the Surface Scanner
- **Create: Diesel Generators** Pumpjack Bearing and all related components, Oil Deposits, and Distillation Controller

# A Note on Create Contraptions
These restrictions do NOT apply to Simulated Contraptions from Create: Aeronautics. <br/>
The following devices will not function when used on a vanilla Create Contraption (this includes trains, sadly):
- Mechanical Saw
- Mechanical Drill
- Deployer
- Harvester
- Andesite and Brass Funnels
<br/> 
Note that this list does not include Chests, Fluid Tanks, and Portable Storage Interfaces. If you abuse this I will change my mind. <br/>
STOP DOING CIRCLE FARMS.

# A Note on Create Trains
Create mod trains will drive incredibly slowly without a source of power. Fuel can be provided as coal in a non-vault storage on the train. Pondering a train station can explain. <br/>
Create mod trains are also powerable with electricity, which drive faster than coal-powered trains. Pondering a pantograph can explain.

In the future, Create mod trains will be replaced with a sublevel implementation from Create: Aeronautics and trains will be directly driven with any source of rotational power, I am waiting for a specific addon to release.
